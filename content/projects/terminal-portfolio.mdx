# Terminal Portfolio

This project is a personal website designed to look and feel like a terminal.

## Features

- **Command Line Interface**: Type commands to navigate.
- **Responsive Design**: Works on mobile and desktop.
- **Customizable**: Easy to theme and add content.

## Tech Stack

- Next.js 14 App Router
- TypeScript for type safety
- Tailwind CSS for styling
- MDX for content

## Usage

\`\`\`bash
npm run dev
\`\`\`
